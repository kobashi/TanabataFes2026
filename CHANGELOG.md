# Changelog

## 1.1.0

- 管理画面の投稿一覧にページャーと投稿詳細表示スイッチを追加。
- 投影設定で表示数、移動量、文字アニメーション速度、ローテーション間隔を調整できるように変更。
- 新規承認された短冊を優先表示し、通常ローテーションは古い表示から順に入れ替えるように変更。
- 文字数に応じてタイプ風描画速度を変え、短いメッセージほど遅く表示するように調整。
- 管理画面から流星群演出を手動発生できるように追加。
- 自動演出スイッチとイベント間隔を追加し、指定間隔で流星群演出を発生できるように変更。
- 流星群演出中は短冊ローテーションと新規導入を一時停止し、演出終了後に通常表示へ戻るように変更。
- 投影画面へのリンクを投稿画面に追加。
- 運用中Nodeと開発用Nodeを切り替えながら開発する手順をドキュメントに整理。
